📊 Sales Performance Dashboard – Power BI Project

📌 Project Overview
This project analyzes sales performance using Excel and Power BI.
The goal of the project is to clean raw sales data, analyze business performance, and build an interactive dashboard for better decision-making.
The dashboard provides insights into:
Total Sales
Total Profit
Total Orders
Sales Trends
Region-wise Performance
Category-wise Analysis
Top Products
Sales Representative Performance

🧹 Data Cleaning Process
The following cleaning steps were performed in Excel and Power BI:
Removed duplicate records
Converted date columns into proper date format
Checked missing values
Standardized text formatting
Verified numerical data types
Created calculated columns:
Profit

📈 Key Business Insights
North region generated the highest sales.
Electronics category contributed the highest revenue.
Some products generated high sales but lower profit margins.
Monthly sales trends showed seasonal fluctuations.
Sales performance varied across sales representatives.
